Allow users to add a movie to their list of favorites

Needs Bearer Token 		/users/[username]/Movies/[movie] POST :Username, :Movie Movie name A text message saying whether the movie has been added.
Allow users to remove a movie from their list of favorites

Needs Bearer Token 		/users/[username]/Movies/remove/[movie] POST :username, :movie None A text message saying whether the movie was successfully removed
Allow existing users to deregister

Needs Bearer Token 		/users/[Username] DELETE :username None A text message saying whether the user was successfully deregistered.
